# Workflow: Idee Cortex

1. Leggi **tutti i file** nella cartella `cortex/idee/`:
   - `feature_inventory.md` — lista completa di TUTTE le feature già implementate
   - `roadmap.md` — idee future, TODO, e feature completate
2. Mostra all'utente le idee attuali divise per stato (Priorità Alta / Media / Nice to Have)
3. Sulla base dell'inventario feature e dello stato attuale del progetto, suggerisci 3-5 nuove idee creative che NON esistono già
4. Per ogni suggerimento, dai una valutazione (🔥) e spiega perché è importante
5. Chiedi all'utente quali vuole aggiungere o sviluppare
6. Aggiorna `roadmap.md` con le nuove idee approvate
7. Se una feature viene implementata, segnala come [x] in `feature_inventory.md`

**IMPORTANTE:** Controlla SEMPRE `feature_inventory.md` prima di suggerire idee per evitare di proporre cose già implementate!
