4. Intelligenza Artificiale e Contenuto Generato

Cortex utilizza Google Gemini per generare flashcard, quiz e piani di studio. Il contenuto AI:

  • È prodotto sul tuo materiale — non da dati di altri utenti
  • Non è usato da Google per addestrare modelli pubblici (API senza data-training opt-in)
  • Potrebbe contenere imprecisioni — verificalo con le fonti originali
  • Non costituisce parere professionale (medico, legale, finanziario)

5. Minori

Cortex è rivolta a studenti di 14-25 anni. Non raccogliamo consapevolmente dati di bambini sotto i 13 anni. Se sei genitore e ritieni che tuo figlio under-13 abbia creato un account, scrivici a francesco1cutugno@gmail.com per la cancellazione immediata.

Vengono eliminati permanentemente: profilo, mazzi, sessioni, XP, Neural Sparks, FCM token. I dati fiscali legati a pagamenti sono conservati 7 anni per obbligo di legge.

9. Sicurezza

  • Tutte le comunicazioni avvengono via HTTPS/TLS 1.3
  • Firebase Security Rules limitano l'accesso ai soli dati dell'utente autenticato
  • Firebase App Check protegge le API da accessi non autorizzati
  • La chiave API Gemini è conservata esclusivamente sul dispositivo tramite localStorage
  • Nessuna password raccolta — autenticazione esclusivamente tramite Google OAuth 2.0

10. Notifiche Push

Cortex può inviare notifiche push tramite Firebase Cloud Messaging (FCM) per: reminder di studio, aggiornamenti funzionalità, eventi speciali di studio. Il consenso è richiesto esplicitamente. Puoi revocarlo in qualsiasi momento dalle impostazioni del dispositivo.

11. Archiviazione Locale (localStorage)

L'app non usa cookie di tracciamento. Usa localStorage esclusivamente per: preferenze utente, chiave API Gemini (solo dispositivo), stato onboarding, cache mazzi offline. Dati eliminati alla disinstallazione.

12. Modifiche a Questa Policy

Notificheremo modifiche sostanziali tramite notifica push o banner in-app almeno 15 giorni prima dell'entrata in vigore. Policy sempre aggiornata su cortexapp.it/privacy.
